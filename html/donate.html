If you like you may also try one of my other extensions:
  • DeZog
    A debugger for Z80 assembler code.
  • Z80 Instruction Set
    Shows the opcode, affected flags and a description on hovering over a Z80 instruction.
  • Hex Hover Converter
    Converts numbers into their decimal, hexadecimal and binary format on hovering.
  • Binary File Viewer
    The 'Binary File Viewer' is a framework that allows you to build your own custom viewers for binary files.
  • Number Plotter
    The "Number Plotter" takes a series of selected numbers and creates a plot out of it.
  • ZX SNA File Viewer
    A viewer for the ZX Spectrum snapshot (.sna) format (in marketplace search for "snafile").
  • ZX NEX File Viewer
    A viewer for the ZX Spectrum Next (.nex) file format (in marketplace search for "nexfile").
